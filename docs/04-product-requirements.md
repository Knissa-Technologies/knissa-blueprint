# 📘 Product Requirements Document (PRD)

> **Knissa MVP v1.0**

---

# Product Overview

Knissa is a digital financial platform designed to make money management simple, secure and accessible.

The first version (MVP) will focus on the essential features required for users to create an account, manage a digital wallet and perform secure money transfers.

---

# Goals

The MVP must allow users to:

- Create an account
- Log in securely
- Manage their profile
- View wallet balance
- Send money
- Receive money
- View transaction history

---

# Target Users

### Individuals

People who need a simple and secure way to manage their money.

### Merchants (Future)

Small businesses that want to accept digital payments.

---

# Functional Requirements

## Authentication

Users can:

- Register
- Login
- Logout
- Reset password
- Verify email

---

## User Profile

Users can:

- View profile
- Update profile information
- Change password

---

## Wallet

Each user has one wallet.

The wallet contains:

- Current balance
- Currency
- Account status

---

## Transactions

Users can:

- Send money
- Receive money
- View transaction history

Each transaction contains:

- Transaction ID
- Sender
- Receiver
- Amount
- Currency
- Status
- Timestamp

---

## Dashboard

The dashboard displays:

- Wallet balance
- Recent transactions
- Quick actions

---

## Notifications

Users receive notifications for:

- Money received
- Money sent
- Login from a new device

---

# Non-Functional Requirements

The platform must be:

- Secure
- Fast
- Reliable
- Scalable
- Responsive

---

# Security Requirements

- JWT Authentication
- Password hashing
- HTTPS
- Role-based authorization
- Audit logs

---

# MVP Scope

Included:

- Authentication
- User Management
- Wallet
- Transactions
- Dashboard
- Notifications

Excluded:

- Virtual Cards
- Loans
- Investments
- International Transfers
- Merchant Portal
- Mobile Application

---

# Success Metrics

The MVP is successful when users can:

- Create an account
- Log in successfully
- Send money
- Receive money
- View their transaction history
- Trust the platform
